# Changelog

All notable changes to `@darkoum/ng-datagrid` will be documented in this file.

## [1.0.0] — 2026-06-03

### Added
- DataGrid component (dx-data-grid replacement) — sort, filter, pagination, inline/popup edit, virtual scroll, CSV/XLSX export
- ThaiDatepicker component — พ.ศ. calendar, 8 formats
- Form component (dx-form replacement) — colCount, group items, all editorTypes
- SelectBox (dx-select-box) — search, CVA
- NumberBox (dx-number-box) — spin, format, CVA
- TextBox (dx-text-box) — clear, password mode, CVA
- CheckBox (dx-check-box) — indeterminate, CVA
- DateBox (dx-date-box) — standalone Thai date wrapper
- TagBox (dx-tag-box) — multi-select, search, CVA
- RadioGroup (dx-radio-group) — vertical/horizontal, CVA
- TextArea (dx-text-area) — maxLength counter, CVA
- Switch (dx-switch) — animation, on/off text, CVA
- Button (dx-button) — 5 types, icons
- Popup (dx-popup) — overlay, ESC, events
- TabPanel (dx-tab-panel) — selection events
- Accordion (dx-accordion) — multiple/collapsible
- LoadPanel (dx-load-panel) — overlay spinner
- ThaiDate pipe — 8 formats + custom pattern
- ResizableColumnDirective — drag-to-resize columns
